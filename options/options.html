About CodeSync

CodeSync

Sync. Showcase. Succeed.

Manifest V3 Chrome Extension MIT License v0.2.0

CodeSync eliminates the manual work of maintaining a solutions repository. It detects accepted submissions from LeetCode and GeeksforGeeks directly in your browser, extracts the code along with metadata (title, difficulty, tags, language), and pushes formatted files to your GitHub repository — all with a single click.

One-click solution capture from LeetCode & GFG
Auto-detects language, difficulty, tags & problem link
Queue-first workflow — nothing is lost offline
Smart file organization with metadata headers
